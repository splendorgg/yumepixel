## Getting Started
Ensure you have [Git](https://git-scm.com/) and [Node](https://nodejs.org/en) (v22.11.0) installed.

#### Clone this repository
```bash
git clone https://github.com/splendorgg/yumepixel.git
cd yumepixel
```

#### Install dependencies
```bash
npm install
```

#### Run the project
```bash
npm run dev
```
